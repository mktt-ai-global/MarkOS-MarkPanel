<h2 id="english">English</h2>

First off, thank you for considering contributing to MarkOS! It's people like you that make MarkOS such a great tool for the NAS community.

### How Can I Contribute?

#### Reporting Bugs
* **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues).
* If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues/new).

#### Suggesting Enhancements
* **Check if it has already been suggested** in the [Issues](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues).
* Explain why this enhancement would be useful to most MarkOS users.

#### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

### Style Guide
* **Backend**: Follow PEP 8 for Python (FastAPI).
* **Frontend**: Use TypeScript, Tailwind CSS, and follow the [Next.js best practices](https://nextjs.org/docs/getting-started/project-structure).
* **Design**: Stick to the "Frosted Glass" (磨砂玻璃) UI tokens defined in `tailwind.config.ts`.

<h2 id="中文">中文</h2>

首先，感谢您考虑为 MarkOS 做出贡献！正是有了像您这样的人，MarkOS 才能成为 NAS 社区的优秀工具。

### 我该如何贡献？

#### 报告 Bug
* **确保该 Bug 尚未被报告**：在 GitHub 的 [Issues](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues) 中进行搜索。
* 如果找不到相关的 Issue，请 [提交一个新的 Issue](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues/new)。

#### 提出改进建议
* **检查是否已有类似建议**：在 [Issues](https://github.com/mktt-ai-global/MarkOS-MarkPanel/issues) 中查看。
* 说明为什么这项改进对大多数 MarkOS 用户有用。

#### 提交合并请求 (Pull Requests)
1. Fork 本仓库并从 `main` 分支创建您的分支。
2. 如果您添加了需要测试的代码，请添加测试用例。
3. 如果您更改了 API，请更新相关文档。
4. 确保所有测试通过。
5. 确保代码符合 Lint 规范。
6. 提交 Pull Request！

### 代码规范
* **后端 (Backend)**: Python (FastAPI) 请遵循 PEP 8 规范。
* **前端 (Frontend)**: 使用 TypeScript, Tailwind CSS，并遵循 [Next.js 最佳实践](https://nextjs.org/docs/getting-started/project-structure)。
* **设计 (Design)**: 必须符合 `tailwind.config.ts` 中定义的“磨砂玻璃”(Frosted Glass) UI 设计规范。
